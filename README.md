## 🚀 Projects

### ⚙️ [Lightweight Rule Engine for Automation](https://github.com/Harry9021/lightweight-rule-engine)
`Go` `React` `Vite` `Tailwind CSS` `SQLite` `gopsutil`

Real-time rule engine for system automation on live CPU/RAM thresholds. Go backend with SQLite persistence, and a React + Tailwind UI for rule management and metric visualization.

---

### 💆 [Skincare Recommendation System](https://github.com/Harry9021/skin-care-recommender)
`React` `Tailwind CSS` `Flask` `scikit-learn` `Google OAuth`

Full-stack recommender using **KNN** trained on **225 products** across 6 categories (cleanser, moisturizer, sunscreen, etc.), achieving **82% accuracy**. Takes skin type + up to 3 concerns and returns top-10 product suggestions via Flask REST API with Google OAuth and a responsive UI.

---

### 🎙️ [Speech Emotion Recognition](https://github.com/Harry9021/Speech-Emotion-Recognition)
`Vite` `React` `Flask` `PyTorch` `Python`

CNN pipeline on **Mel spectrograms** achieving **78%+ accuracy** on the RAVDESS dataset (8 emotion classes). Flask API with sub-2s latency and a Vite + React interface for real-time emotion visualization.

---

### 🔍 [Lost and Found Platform](https://github.com/Harry9021/LF-project)
`React.js` `Node.js` `Express.js` `MongoDB`

Web platform to report and locate lost items with secure JWT authentication, a responsive UI, and real-time search across item listings.
