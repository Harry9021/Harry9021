### 💆 Skincare Recommendation System · `Flask · scikit-learn · React · Google OAuth`

> **KNN model** trained on 225 products across 6 categories (cleanser, moisturizer, sunscreen, etc.) — **82% accuracy**. Takes skin type + up to 3 concerns → returns top-10 product suggestions via REST API. Includes Google OAuth, responsive frontend, and a product dataset I cleaned from scratch.

📦 [`github.com/Harry9021/skin-care-recommender`](https://github.com/Harry9021/skin-care-recommender)

---

### 🎙️ Speech Emotion Recognition · `PyTorch · Flask · Vite · React`

> CNN pipeline on **Mel spectrograms**, achieving **78%+ accuracy** on RAVDESS (8 emotion classes). Flask API handles inference with sub-2s latency. Built because I wanted to see if audio alone could tell a story — turns out it really can.

📦 [`github.com/Harry9021/Speech-Emotion-Recognition`](https://github.com/Harry9021/Speech-Emotion-Recognition)
